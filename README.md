Dry Eye Defense

A fast-paced pixel-art action game about surviving waves of germs using tear power.

Play here: https://kingdonut92.github.io/DryEyeDefense/

Controls
- Move: A/D or Arrow Keys
- Jump: Space or J (mobile)
- Shoot: J or S (mobile)
- Shop: B
- Pause: P or Esc

Built with pure HTML5 Canvas + JavaScript

this is v0.1, first playable public build.

I am currently designing all the audio, which will be patched into the game soon!
